# Mintlify Website Clone (HTML & CSS)

## Overview
This project is a static clone of the Mintlify marketing website, built using only HTML and CSS.  
The goal of the project was to practice layout, spacing, typography, and modern SaaS-style UI patterns without using JavaScript or frameworks.

## Sections Recreated
The following sections were recreated from the Mintlify website:

- Navigation bar (header)
- Hero section
- Logo wall / customers section
- Intelligence features section
- Enterprise section
- Customer stories section
- Final call-to-action (CTA) section
- Footer

All sections focus on visual structure and layout rather than interactivity.
